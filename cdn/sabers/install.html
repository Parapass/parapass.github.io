    How to Install Sabers

    This assumes you have already modded your game. Don't know? Check BSMG's guide on how to mod your game.

    PC

    To install PC sabers, find your Beat Saber install folder (usually C:\Program Files (x86)\Steam\steamapps\common\Beat Saber if using Steam, or in C:\Program Files\Oculus\Software\Software\hyperbolic-magnetism-beat-saber).

    The folder you need to drag the .saber files to is called CustomSabers. If you're using ModAssistant OneClickâ„¢ Install, then the sabers will be automatically installed for you.

    Once the sabers are in the folder, launch the game and check SaberFactory/CustomSabers in the Mods menu to the left and you will have the sabers you've downloaded.

    Quest

    To install Quest sabers, go to the BMBF web portal interface (usually https://192.168.1.(number):50000) and go to the Upload tab.

    Once in the Upload tab, drag and drop/select the saber from its location on your PC or phone.

    In the top right, press Sync to Beat Saber.
